#字符串
***
>**定义**:字符串就是0个或多个排在一起的字符,放在单引号或双引号之中
>>'abc'
>>
>>"abc"

###1. length属性和chatAt()方法
- length属性返回字符串的长度.

	```js
		var s = "hello";
		s.length; // 5
	```
- chatAt()方法
	
   ```js
		返回字符串的第 n 个字符，如果不在 0~str.length-1之间,
		则返回一个空字符串。
		
		var s = "hello";
		s.chatAt(0); // h
		
		注意:因为字符串可以视为数组,所以我们可以使用数组的方括号运算符
		实现和chatAt()一样的效果.
		var s = "hello";
		s[0]; // h
		s[1]; // e
		s[2]; // l
	```


```
同上

str.substr(2,3));

(5)str.substring(2));
截取字符串.从下标为2开始(包含下标2)截取整个字符串.

从下标为2开始,截取到下标为7的元素,不包含下标7 [2,7)

(6) var str2 = str.split("");  不常用
```
